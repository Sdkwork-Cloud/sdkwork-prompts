# SDKWork Local Workspace

This source-controlled `.sdkwork/` directory stores repository-local development knowledge and agent extensions for `sdkwork-prompts`.

Authoritative directories:
- `.sdkwork/skills/`: repository/application skills.
- `.sdkwork/plugins/`: repository/application agent plugins.

This directory must not contain runtime state, generated SDK output, caches, secrets, databases, or user-private files.
