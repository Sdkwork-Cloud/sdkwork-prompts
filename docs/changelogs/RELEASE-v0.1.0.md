# sdkwork-prompts v0.1.0 (pre-release)

## Capabilities

- Intelligence prompts registry with six `ai_` tables
- 18 HTTP routes: app, backend, and open API surfaces
- `SdkWorkApiResponse` / `ProblemDetail` on all L2+ JSON responses
- PC workspace (`apps/sdkwork-prompts-pc`) via `@sdkwork/prompts-backend-sdk`
- Docker Compose: API + PostgreSQL
