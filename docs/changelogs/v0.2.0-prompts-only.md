# v0.2.0 — Prompts-only alignment

## Summary

Removed forum (`prm_`) capability. Application is now exclusively the intelligence **prompts** registry.

## Database

- 6 tables: `ai_prompt_category`, `ai_prompt`, `ai_prompt_version`, `ai_prompt_binding`, `ai_prompt_template`, `ai_prompt_usage`
- Renamed: `ai_agent_prompt_template` → `ai_prompt_template`, `ai_prompt_usage_event` → `ai_prompt_usage`

## API

- 18 routes (was 66 forum + prompts mix)
- Agent template endpoints wired in runtime

## Removed

- Forum crates (service, repository, worker, route descriptors)
- `specs/prompts-database.schema.yaml` (44 `prm_` tables)
- Forum docker worker, search sidecar defaults
